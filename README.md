# Boston Crime Data Analysis

This project explores Boston crime incident data to identify patterns and trends related to crime types, locations, and time periods.

## Project Goals

* Analyze the most common crime types
* Explore changes in crime over time
* Identify districts with higher numbers of reported incidents
* Create charts and visualizations of the findings

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Visual Studio Code

## Dataset

The dataset contains reported crime incidents in Boston, including details such as offense type, date, time, district, and location.

## Running the Project in VS Code

1. Open the project folder in VS Code.
2. Install the **Python** and **Jupyter** extensions.
3. Open the `.ipynb` notebook file.
4. Click **Select Kernel** at the top of the notebook.
5. Choose your Python environment.
6. Run each cell using the play button.

Install the required packages in the VS Code terminal:

```bash
pip install pandas matplotlib jupyter
```

## Disclaimer

This project is for educational and analytical purposes. The dataset represents reported incidents and may not include all crime activity in Boston.
